Analisi DUSAF 7 Comune Lombardo

Plugin QGIS per l'analisi automatizzata dell'uso del suolo nei Comuni lombardi
tramite dataset DUSAF 7.0 e confini amministrativi ISTAT 2026.

Il plugin registra un provider Processing dedicato e un algoritmo per:
- validare i layer richiesti DUSAF7 e Com01012026_WGS84;
- eseguire il workflow di analisi per Comune lombardo;
- produrre GeoPackage, CSV riepilogativo e audit QC-4.
