Analisi DUSAF 7 Comune Lombardo

Plugin QGIS per l'analisi automatizzata dell'uso del suolo nei Comuni lombardi tramite dataset DUSAF 7.0 e confini amministrativi ISTAT 2026.

Il plugin registra un provider Processing dedicato e un algoritmo per validare i layer richiesti, eseguire il workflow DUSAF per Comune lombardo e produrre GeoPackage, CSV riepilogativo e audit QC-4.
